SET UP:
1. Have Node.js installed 
2. Run npm install --save dotenv in server folder, place .env file into the server folder 
3. Run npm install express mongoose cors nodemon in server folder if needed 
4. Run npm install bootstrap axios react-router-dom react-bootstrap in client folder if needed 
5. The client folder template was created with npx create-react-app ${PROJECT_NAME}
6. Please have 2 terminals run in VSCODE, cd server > node index.js, cd client > npm start to run frontend + backend
